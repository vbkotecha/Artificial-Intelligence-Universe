# Security Policy

## About This Repository

Artificial Intelligence Universe is a **curated catalog** of AI tools and resources. It contains no running code, no dependencies, and no executable software. The repository consists entirely of Markdown documentation and static configuration files.

Despite this, we take certain security concerns seriously — especially those that could harm users of this catalog.

---

## Reporting a Vulnerability

**Do NOT report security concerns via public GitHub issues.** Public disclosure of security issues before they are addressed puts users at risk.

Instead, email the maintainer directly:

> **frangelrcbarrera@gmail.com**

Please include:
- A clear description of the issue
- Steps to reproduce (if applicable)
- The specific URL or entry in question
- Any supporting evidence (screenshots, logs, etc.)

---

## Response Times

| Severity | Acknowledgment | Initial Assessment |
|----------|---------------|-------------------|
| Critical | Within 24 hours | Within 48 hours |
| High | Within 48 hours | Within 7 days |
| Medium / Low | Within 48 hours | Within 7 days |

These timelines apply to the maintainer's best effort and may vary depending on availability.

---

## What We Take Seriously

### Malicious Links
If a link in the README redirects to malware, phishing sites, scams, or otherwise harmful content, report it immediately as a **Critical** issue.

### Spam / Scam Tools
If a listed tool is discovered to be fraudulent (e.g., a fake AI service collecting payments without delivering a product), it should be removed. Report as **High** severity.

### Account / Repository Takeover
If you suspect the maintainer's account or this repository has been compromised, report as **Critical**. Use alternative contact methods if email is unavailable.

### Supply Chain Concerns
While this repo has no dependencies, if a listed tool's website is serving malicious downloads or the tool's GitHub repository has been compromised, report as **Medium** severity.

---

## What Is Out of Scope

This security policy applies **only to the Artificial-Intelligence-Universe repository itself**. It does not cover:

- Security vulnerabilities in the external tools, platforms, or services listed in the README. Contact those projects directly.
- General questions about the safety or reliability of listed AI tools (use Discussions or Issues for that).
- Phishing or scam reports unrelated to this repository.

---

## Responsible Disclosure

We ask that you:

1. Report the issue privately (see [Reporting a Vulnerability](#reporting-a-vulnerability))
2. Allow reasonable time for the maintainer to investigate and address the issue before any public disclosure
3. Do not exploit the issue beyond what is necessary to demonstrate it
4. Do not access, modify, or delete data that does not belong to you

We will acknowledge contributors who responsibly disclose valid security issues (with their consent).
